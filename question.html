What is Semantic Web?

The Semantic Web is an extension of the World Wide Web to enable machines to understand the meaning of information on the web.

Answers

The Semantic Web uses technologies like RDF and OWL to represent data in a structured and meaningful way.

It aims to convert the existing web of documents into a web of data which can be processed by machines.
